Usage Instructions

NOTE: [] brackets denote optional step. These options are not documented, explore at your own risk or enjoyment. :-)

  1. Browse for image source.
  2. [Click View Input Image, if you want to compare with ASCII output.]
  3. [Select output options.]
  4. [Select ASCII text options.]
  5. [Select color options.]
  6. [Select output size, scale, and format options.]
  7. Click ASCII-fy Image to generate ASCII version of image.

Default Settings

The following settings are the defaults when you do not make a selection for any of the optional options mentioned in the instruction steps in the previous section.

  • ASCII output saved to source image directory (output saved with same name but ends with .txt or .htm extension)
  • Generate ASCII with alphanumeric characters
  • Font size set to 6 point
  • Background color set to White
  • Foreground ASCII text set to Black
  • ASCII output format set to HTML
  • ASCII color option set to multiple/gradient forecolor text via HTML
  • ASCII output size based on 100 x 100 pixel width & height
  • Anti-alias scale down factor set to 1
